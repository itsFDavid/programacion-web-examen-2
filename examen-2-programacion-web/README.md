## Pasos

1. Clonar el repositorio

```bash
git clone https://github.com/itsFDavid/programacion-web-examen-2.git
```

2. Entrar a la carpeta del examen

3. Instalar las dependencias

```bash
pnpm install

npm intall
```

4. Ejecutar el proyecto en modo dev

```bash
# escoje la opcion
pnpm run dev

npm run dev
```
